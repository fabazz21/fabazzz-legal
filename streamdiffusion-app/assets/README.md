# Assets

This folder contains demo assets and generated images.

## Demo

To see the app in action:

1. **Text-to-Image Demo**: Run `./run_txt2img.sh` (or `run_txt2img.bat` on Windows)
2. **Image-to-Image Demo**: Run `./run_img2img.sh` (or `run_img2img.bat` on Windows)

## Screenshots

After running the app, you can save screenshots here for documentation.

## Generated Images

By default, generated images will be stored in memory. You can modify the code to save them to this directory.
